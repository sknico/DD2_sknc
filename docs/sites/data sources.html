Data structure

DENNE SIDE SKAL GIVE ET OVERBLIK OVER DE FORSKELLIGE DATAKILDER. DER SKAL INKLUDERES NOGLE FIGURER.

DD2 is based on data from multiple sources:

(vi skal have nogle gode navne til de forskellige datakilder)

  • DD2 core data, i.e., data that makes the foundation of DD2

    • DD2 questionnaire data

    • Biomarker data (analysis results based on the blood and urine samples from time of enrollment)

  • Additional DD2 data, e.g., from additional questionnaires and/or substudies. This includes data only available through DD2, but it was not collected as part of the initial enrollment process.

    • IDNC

    • Fødselsdata

  • Data extracts from external data sources

    • DDDA (udgår)

    • Fodstatus

    • Diabase (øjenkompl.)

  • Danish medical registry data

    • SDS

    • DST
